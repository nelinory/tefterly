Notes Features:
===============
[X] Rich text font styles: Bold/Italic/Underline/Strikethrough and Highlight
[X] Rich text paragraph styles: Left/Right/Center/Justify align, Increase/Decrease indent and Bullet/Ordered list
[X] Note operation: Starred/Unstarred, Delete/Restore/Permanently Delete, Archive/Restore, Duplicate and Print
[X] SpellCheck
[X] Search across all notes
[X] Search in note
[X] Dynamically formatted hyperlinks
[X] Error handling and logging
[X] Automatic notes backup at specified interval
[X] Automatic notes backup on startup if the version of the app settings is different since the last run
[ ] Dark/Light theme support
[ ] Attach categories to a note
[ ] Ability to resize image
[ ] Export note to: RTF, HTML and TXT
[ ] Encrypt/decrypt notes with a password
